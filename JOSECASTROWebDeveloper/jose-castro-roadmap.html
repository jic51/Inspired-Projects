Phase One

Get Paid for
What You Already Know

You already know enough to get paid. This phase is about finishing your portfolio, closing your first 2–3 real clients, and building processes that make delivery smooth.

⏱ 0 – 3 Months · Goal: 3 paid projects
Polish These Skills First
CSS Layout Mastery Must

Flexbox and CSS Grid fluently. You should be able to build any layout in your head without googling it every time. This is the #1 thing that separates fast devs from slow devs.

Resources
CSS Grid Garden — game (free, 2h)
Flexbox Froggy — game (free, 1h)
Kevin Powell on YouTube — best CSS teacher
Practice time: ~10 hours · Build 3 different layouts from scratch
CSS Variables & Design Tokens Must

Using --variables for colors, fonts and spacing. This is how you build sites that are easy to update and look consistent throughout. Your clients will ask for color changes — this makes it a 2-minute fix.

Resources
MDN CSS Custom Properties (free docs)
Watch Kevin Powell's "CSS Variables" video
Practice time: ~4 hours · Refactor the Cristina Vera site to use variables
Responsive Design Must

Every site must look perfect on mobile. 70%+ of clients' customers will visit on a phone. Master media queries, fluid typography (clamp()), and mobile-first methodology. Non-negotiable.

Resources
Google's "Responsive Web Design" course (free)
Practice: Open DevTools and test EVERY project on 3 screen sizes
Practice time: ~8 hours · Make 2 existing projects fully responsive
JavaScript DOM Basics Must

querySelector, addEventListener, classList, simple animations. You need to be able to build: hamburger menus, scroll-triggered effects, counter animations, FAQ accordions, image sliders. You already have most of this — deepen it.

Resources
javascript.info — best free JS reference
Recreate the counter animation from the Cristina Vera site from memory
Practice time: ~12 hours · Build each UI component listed above
Git & GitHub Pages Must

Version control + free hosting. Every project goes on GitHub. GitHub Pages = free hosting. You already have a GitHub — now learn to commit, push, and deploy from the command line. Also learn Netlify (even easier free hosting with custom domains).

Resources
Learn Git Branching — interactive game (free)
Netlify docs — deploy in 2 minutes with drag & drop
Practice time: ~5 hours · Deploy your portfolio to a live URL today
Google Fonts & Typography Important

Typography is 80% of design. Learn how to pair fonts (serif + sans-serif), set proper line-height and letter-spacing, and use font weights strategically. This is what makes your sites look premium vs cheap.

Resources
Google Fonts — free, huge library
Fontpair.co — curated font pairings
Study the 5 design variations you already have
Practice time: ~3 hours · Redesign one project using 2 different font pairings

Phase Two

Skills That
2x Your Rates

These skills will take your pricing from $150–$300 to $400–$900 per project. Each one adds real, tangible value that clients will pay more for.

⏱ 3 – 6 Months · Goal: $500+ average project
EmailJS / Formspree Pays for Itself

Contact forms that actually send emails — without any backend/server. EmailJS is free up to 200 emails/month. This single skill lets you charge $100 more per project and every client needs it.

Resources
emailjs.com — official docs + quickstart (1 day)
formspree.io — even easier alternative
Time to learn: ~1 day · Immediate revenue impact
CSS Animations & Scroll Effects High Visual Impact

@keyframes, transitions, IntersectionObserver for scroll-triggered reveals. This is what makes sites look like they cost $3,000 when they cost $500. Study the 5 design variations you built — those animations are already in the code.

Resources
Animate.style — CSS animation library reference
MDN IntersectionObserver docs
Study the scroll reveal code in your Cristina Vera variations
Time to learn: ~1 week · Add to all new projects
SEO Basics Clients Love This

Meta tags, Open Graph (for social previews), structured data basics, fast load times, semantic HTML. You don't need to be an SEO expert — just doing the basics puts you above 80% of competitors and justifies $100+ extra per project.

Resources
Google's "SEO Starter Guide" (free, official)
Ahrefs free SEO course on YouTube
PageSpeed Insights — test every project
Time to learn: ~3 days · Template it into every project
Google Analytics Setup Easy Upsell

30 minutes to learn, adds real value. Set up GA4 on every project. Clients love seeing how many visitors they have. Charge $50–$100 extra for this and it takes you 30 minutes per project.

Resources
Google Analytics Academy (free official course)
Time to learn: ~2 hours
Figma Basics Professional Workflow

Design mockups before coding. Clients can approve the design before a single line of code is written. This eliminates scope creep and "I don't like how it looks" complaints. Figma is free.

Resources
Figma official tutorials (free on YouTube)
Figma for Beginners — Figma's own free course
Time to learn: ~1 week · Basic wireframing only needed
Basic Performance Optimization SEO + UX

Image compression (Squoosh.app), lazy loading, minification. Google ranks faster sites higher. Learn to get a 90+ score on Google PageSpeed Insights. This directly affects your client's Google ranking.

Resources
squoosh.app — free image compression
web.dev — Google's performance guides (free)
Time to learn: ~1 day

Phase Three

International
Market Access

The US/Spain market pays 3–5x more than Ecuador for the same work. These skills get you there — React, Tailwind, Upwork, and professional English communication.

⏱ 6 – 12 Months · Goal: First international client
React Basics Industry Standard

Components, props, state, useState, useEffect. You don't need to be a React expert — just enough to build landing pages and multi-page apps. React is the most-requested framework on Upwork by far. Coming from JS, it clicks fast.

Resources
React official docs — react.dev (completely rewritten, excellent)
Scrimba React course (free + interactive)
Goal: Rebuild one of your HTML sites in React
Time to learn basics: ~3–4 weeks
Tailwind CSS Speed Multiplier

Utility-first CSS framework. Build beautiful UIs 3x faster than writing raw CSS. Almost every US company uses it. Used in this very portfolio (Claude artifacts use Tailwind). Once you learn it, you won't go back.

Resources
tailwindcss.com/docs — best docs in web dev
Tailwind CSS YouTube playlist by official team
Time to learn: ~1 week
Upwork Profile Setup Gate to US Market

100% complete profile, portfolio images, compelling bio. The first 5 reviews are everything — price low to get them ($5–$15/hr to start), then raise to $25–$50/hr. Most Latin American devs charge $15–$35/hr on Upwork and work consistently.

Resources
Upwork's official "How to get started" guide
YouTube: "Upwork profile 2025 tips"
Setup time: ~2 days · First proposal within week 1
Technical English Required for USA

You don't need perfect English — you need professional written English for emails and proposals. And basic conversational for Zoom calls. Focus on: email etiquette, proposal writing, and explaining technical decisions clearly. Tools help: Grammarly, ChatGPT for email drafts.

Resources
Grammarly — free browser extension
Watch US dev YouTubers — absorb tone and style
Your portfolio is already in English — great start
Ongoing skill — practice daily

Strategy

How to Get
Paying Clients

The skills are only half the equation. This is how you turn them into money — locally, nationally, and internationally.

🏃
The Door-to-Door Pitch
Walk into local businesses with your phone showing your portfolio. Say: "I searched for your business online and couldn't find a website. I build sites for businesses like yours. Can I show you something?" Close rate is surprisingly high with warm leads. Target: dentists, barbers, lawyers, restaurants, gyms, tutors.
👥
Network in Your Circle
Tell everyone you know that you build websites. Post your work on Instagram and Facebook. Your first 5 clients will likely come from people you already know. Don't be shy about it. Post "before & after" screenshots — they go viral in local communities.
💬
Facebook Groups
Join local business groups and offer free value (tips, feedback on websites). Don't spam — provide genuine help first, mention your services naturally. This builds trust and inbound leads. Also join "needing a web developer" type groups nationally.
🌐
Upwork & Fiverr (International)
Create profiles on both platforms. On Fiverr: list specific gigs ("I will build a professional landing page for your law firm"). On Upwork: apply to 5–10 jobs per week with personalized proposals. First month: aim for reviews, not money. Reviews = everything.
📸
Instagram/TikTok Content
Post your work. Before/after redesigns. "I built this in 3 days" posts. Time-lapses of you coding. Behind-the-scenes. This builds a following that becomes inbound clients. Takes 3–6 months to work but it's 100% free and compounds forever.
🔄
Monthly Maintenance Pitch
After every project, offer monthly maintenance for $50–$100/month. Frame it as: "I'll make sure your site stays updated, fast and secure." Even if it only takes 1 hour a month, the recurring income adds up fast. 10 clients = $500–$1000/month passively.
